Ansible安装和简单使用

Written by Menah3m with ♥ on August 31, 2020 in

  • 测试环境

    操作系统：CentOS 7

    Ansible客户端：centos-master（192.168.13.100）

    节点：centos-slave1（192.168.13.101） centos-slave2（192.168.13.102）

  • 安装步骤

    #设置EPEL仓库
  
[root@centos-master ~]# wget <http://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm>
[root@centos-master ~]# rpm -ivh epel-release-latest-7.noarch.rpm
#检查EPEL仓库是否存在
[root@centos-master ~]# yum repolist -v    
  
#安装Ansible
[root@centos-master ~]# yum -y update
[root@centos-master ~]# yum -y install ansible
#检查Ansible版本
[root@centos-master ~]# ansible --version
  
#设置SSH密钥
[root@centos-master ~]# ssh-keygen
[root@centos-master ~]# ssh root@192.168.13.101
[root@centos-master ~]# ssh root@192.168.13.102
  
#定义节点清单
[root@centos-master ~]# vi /etc/ansible/hosts
  
#添加以下内容
 [slavenodes]
 192.168.13.101
 192.168.13.102

  • 简单使用

    #测试Ansible服务端和节点之间能否通信
[root@centos-master ~]# ansible -m ping "slavenodes"
